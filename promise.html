우리는 상상을 현실로 구현합니다.
Promise는 교내 유일 웹 개발 동아리입니다.
Promise와 함께라면 당신의 상상은 현실이 됩니다.

소개

Promise는 교내 유일 웹 개발 동아리이며,
웹과 서버 개발을 배우고 함께 실습하며
웹 개발자로서의 역량을 쌓아나가는 동아리입니다.

커리큘럼

웹 개발 기초
웹의 작동 방식과 웹을 이루는 언어인
HTML, CSS 등을 학습합니다.
프론트엔드 기초
프론트엔드 웹 개발의 기반인
JavaScript를 통해 동적 웹을 만듭니다.
프론트엔드 심화
ReactJS를 통해 더욱
효율적으로 웹을 개발합니다.
서버 개발
TypeScript를 통한 백엔드 개발을 배워
웹 서버를 구현합니다.
